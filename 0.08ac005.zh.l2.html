帕里奇亚海湾

  • font size decrease font size increase font size
  • Print
  • Share
帕里奇亚海湾附近的小路非常漂亮，与海岸线平行。从港口开始北行。穿过植有海松的小海滩，踏浪前行，前往帕里奇亚的长滩利瓦迪亚(Livadia)。离开海滩，沿着沙滩上方的土路行进，直至看到几栋房屋，就抵达利瓦迪亚海滩的尽头。
穿过狭窄的沙滩，您将发现自己处于与海岸线平行的岩石小路上，右侧有几座房屋。沿着小路走约500米到达第一个海角。小路在到达悬崖边缘之前偏离了海岸线，向西朝着克里奥斯(Krios)和马瑟罗(Marcello)海滩延伸。

小路此时仍然高于海平面很多，接下来是通往一个小沙滩的下坡路。沙滩一端有一个小码头和一个小吃店。您可以在那里停下来喝杯咖啡，吃点东西再继续。远处狭长的沙滩和海松已经遥遥在望。沿沙滩行走，穿过克里奥斯、马瑟罗和卡米尼(Kamini)海滩。  

您会与来自帕里奇亚的道路会合。沿着柏油路末端的土路向圣佛卡斯(Agios Fokas)行进。土路在大约200米后转向内陆。离开此路，沿着悬崖上方和别墅下面的小路行走。小路的尽头是圣佛卡斯小教堂及海滩。
