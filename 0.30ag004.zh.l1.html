LTE Concept

Issue Time：2011-06-21 14:24:24

3GPP Long Term Evolution (LTE) was the largest new technology R&D project launched by 3GPP in the past 2 years. LTE is not the wrongly understood 4G technology, but the transitional stage between 3G and 4G technologies. It is the standard for 3.9G. It improves and enhances the 3G air access technology by using OFDM and MIMO as the only criteria for wireless network evolution. The main performance index of 3GPP LTE project mainly include: provide peak rate of downstream of 100Mbps, upstream 50Mbps in the spectrum bandwidth of 20MHz; improve the performance of community edge subscribers; enhance community capacity; reduce system latency. If the internal singe transmission latency in subscriber plane is less than 5ms, the migration time of control plane from dormancy to activation is less than 50ms, migration time from the residence state to activation is less than 100ms; it supports the community coverage of 100Km radius; able to offer > 100kbps access services fir 350Km / h high speed mobile subscribers; supports paired or unpaired spectrum, and can flexibly configure diverse bandwidths from 1.25 MHz to 20MHz.

3GPP has detailed description for system property requirements, network deployment scenarios, network structure and service supporting ability. Compared to 3G, LTE possesses the following technical characteristics:
(1)The communication rate has greatly improved. The downstream peak rate is 100Mbps and upstream one is 50Mbps.
(2)The spectral efficiency is improved. The downlink is 5(bit/s)/Hz (triple or quadruple of R6HSDPA); the uplink is 2.5(bit/s)/Hz (twice or triple of R6HSU-PA2—3)
(3)The team domain service is the main target. The comprehensive structure of the system is based on team exchange.
(4)QoS ensures that through system design and strict QoS mechanism, the service quality of real time service like VoIP can be warranted.
(5)The system deployment is so flexible that it can support the diverse system bandwidths of 1.25MHz-20MHz and the paired and unpaired spectral deployment. The future system deployment flexibility is thus ensured.
(6)The wireless network delay is reduced. The subframe lengths of 0.5ms and 0.675ms effectively solve the backward compatibility problem and reduce the network delay, which can reach U-plan<5ms and C-plan<100ms.
(7)The community edge byte rate is increased. When the current base station location is maintained, the community edge byte rate is increased. For instance, MBMS(Multi Broadcast Multicast Service) can offer the data speed of 1bit/s/Hz on the edge of community.
(8)Backward compatibility is emphasized. It supports the collaborative operation of current 3G system and non-3GPP standard system.

LTE Advanced Technology will further increase future network speed and capacity for future network society. This technology complies with the global 3GPP Release 10 standards. Recently, LTE Advanced technology has been adopted by International Telecommunication union     (ITU) as one standard that can satisfy international mobile telecommunication standards. The initial LTE Advanced technology is expected to operate in 2013.
LTE Advanced technology’s Enhanced function includes carrier polymer and Multiple Input Multiple Output (MIMO) function. From the perspective of subscribers, this implies that even in the network congestion, the information retrieval and delivery speed will still greatly enhance. With the increasing network speed, subscribers’ experience feeling can thus hugely enhance.
