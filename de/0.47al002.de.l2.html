Begrüßung Outreach Afrika

Auf dieser Seite finden Sie eine Auswahl an Fotos der Begrüßung von Vertretern der Länder Nigeria, Ägypten, Südafrika, Algerien, Senegal sowie Ghana. Über die Navigation können Sie Fotos zu speziellen Themen gezielt ansteuern. Die Bilder des BPA stehen Ihnen zum kostenlosen Download zur Verfügung.

Durch Klick auf ein Foto erhalten Sie eine vergrößerte Ansicht und den Link zum Download. Bitte beachten Sie dabei die Nutzungsbedingungen des BPA.
