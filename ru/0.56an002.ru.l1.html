Conference Server Information

In
order to have a trouble free experience with this video conference
server you should make sure you meet the following minimum system
requirements:

  1. Use of Firefox version 39.0 or later or Chrome version 43.0 or later or Internet Explorer version 11 or later.
  2. Install the latest version for Adobe Flash Player from: https://get.adobe.com/flashplayer
  3. Install the latest version of Java from: http://java.com/en/
  4. Finally your computer should have a Camera with Microphone or a separate Microphone and it should be fully operational.

Before you join any of the conference meetings on this server all of the above should be fully working.
